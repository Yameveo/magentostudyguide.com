Describe Magento skin and javascript files location

 

Skin folder includes Javascript, CSS and image files that are used in .phtml files in template folder, i.e. all the files in this folder vary from theme to theme. From the given picture you can see that there are three main folders â€“ CSS for style-sheets of your theme, images (for media files that are used in your theme) and JS for the templates javascript files.

 

JS folder embodies JS files, libraries and frameworks that are used via frontend and backend. If you need to add a new Javascript/AJAX library or your module from Local code pool requires special scripts, they should be placed here.
